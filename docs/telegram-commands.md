# Telegram Commands

## Session / Context Management

- `/clear` - clear context window.
- `/tokens` - display current context/token usage.
- `/reset` - rebuild agent container from a fresh image.

## Control Modes

- `/takeover` - toggle human takeover mode (XML-to-system control).

## Introspection / Export

- `/give_system_prompt` - send agent system prompt as text file.
- `/give_context` - send full current context window as text file.
