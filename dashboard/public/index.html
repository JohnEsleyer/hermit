Custom Domain Setup

Set up custom domains for the Dashboard and AI agents to receive Telegram webhooks. HTTPS will be automatically provisioned using Let's Encrypt.

Setup Instructions

  1. Point your domain's A record to this server's public IP address.
  2. For agents, you can use a wildcard subdomain (e.g., *.yourdomain.com) or individual subdomains.
  3. Ensure ports 80 and 443 are open on your server's firewall.
